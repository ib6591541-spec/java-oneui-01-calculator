# Калькулятор One UI (Java Swing)

Проект демонстрирует приложение **«Калькулятор»** на Java Swing в едином визуальном стиле, вдохновлённом Samsung One UI.

## Возможности
- Русскоязычный интерфейс и сообщения.
- UI-kit в `src/oneui`:
  - `OneUiTheme` — цвета, шрифты, отступы, радиусы.
  - `OneUiComponents` — скруглённые карточки, кнопки и поле ввода.
  - `OneUiFrame` — базовое окно с большим заголовком.
  - `OneUiDialogs` — русские информационные/ошибки/подтверждения.
- Экран калькулятора:
  - Кнопки `0-9`, `.`, `+`, `−`, `×`, `÷`, `=`, `Очистить`, `←`.
  - Запрет двух точек в одном числе.
  - Обработка деления на ноль: `Ошибка: деление на ноль`.

## Структура
- `src/Main.java`
- `src/app/CalculatorScreen.java`
- `src/app/CalculatorLogic.java`
- `src/oneui/*`

## Запуск
```bash
javac -d out $(find src -name "*.java")
java -cp out Main
```
